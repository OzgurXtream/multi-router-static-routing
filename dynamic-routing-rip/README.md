# Dynamic Routing - RIP

## 📌 Description
RIP protocol is used for automatic route exchange between routers.

## 🔧 Configuration Example

router rip
version 2
network 192.168.0.0


## 🧪 Result
- Routes learned dynamically
- Routing table shows "R" entries
