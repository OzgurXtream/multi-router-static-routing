# Multi Router Network Project (Cisco Packet Tracer)

This project is a complete network simulation built using Cisco Packet Tracer. It demonstrates routing concepts from basic static routing to dynamic routing protocols and inter-protocol redistribution.

---

## 🎯 Project Objective

To design and simulate a multi-router network in order to understand:

- Static Routing fundamentals
- Dynamic Routing protocols (RIP, OSPF, EIGRP)
- Route redistribution between different routing protocols
- Real-world network behavior in enterprise-like topology

---

## 🧱 Project Structure

### 1. Static Routing
Basic routing configuration between routers without dynamic protocols.

📁 Folder: `static-routing/`

---

### 2. RIP Routing
Dynamic routing using RIP protocol to automatically exchange routes.

📁 Folder: `rip-routing/`

---

### 3. OSPF & EIGRP Redistribution
Advanced section demonstrating communication between two different routing protocols using redistribution.

📁 Folder: `OSPF-EIGRP-Redistribution/`

---

## 🔁 Redistribution Concept

Redistribution is used to share routing information between different routing protocols (OSPF ↔ EIGRP). This is typically configured on border routers.
