# Multi Router Network (Static Routing)

## 📌 Overview
This project demonstrates a multi-router network built using Cisco Packet Tracer.

Two different LANs (Kadıköy and Üsküdar) are connected using static routing.

## 🌐 Network Details
- Kadıköy LAN: 192.168.1.0/24
- Üsküdar LAN: 192.168.6.0/24

## 🔧 Features
- Static routing (ip route)
- Multiple routers
- Subnetting
- Connectivity testing with ping

## 🧪 Verification
Routing table and successful ping tests confirm connectivity between networks.
